# Requirements
- JDK 21+

# QUICK START

## Gradle
```
implementation('me.saro:jwt:6.0.0')
```

## Maven
``` xml
<dependency>
  <groupId>me.saro</groupId>
  <artifactId>jwt</artifactId>
  <version>6.0.0</version>
</dependency>
```

## Support Algorithm
| ES | ES256 | ES384 | ES512 |
|----|-------|-------|-------|
| RS | RS256 | RS384 | RS512 |
| PS | PS256 | PS384 | PS512 |
| HS | HS256 | HS384 | HS512 |
